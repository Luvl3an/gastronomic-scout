# Gastronomic Scout

Web application for finding restaurants by mood in Moscow.

## Features
- Search restaurants by mood keywords
- Interactive map with Leaflet.js
- Quick mood suggestions
- Responsive design

## Technologies
- Python, Flask
- HTML, CSS, JavaScript
- SQLite3
- Leaflet.js

## Quick Start
```bash
pip install -r requirements.txt
python init_db.py
python app.py

Open http://localhost:5000
